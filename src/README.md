# Smart_Calc

 ## Возможности

- Вычисление математических выражений
- Вычисление с одной переменной 'x'
- Построение графиков
- Кредитный калькулятор
## Использование
- Ограничение вводимой строки 255 символов
- Если строка является некорректой, калькулятор вернет ошибку
- Для быстрого ввода длинных операторов можно использовать шорткаты (^+...)
## Установка

Перед началом установки убедитесь что у вас установлены следующие пакеты

> Требуется: `GCC` `G++` `MAKE` `CMAKE` `QT6` 
 
- Из папки SRC выполните следующую команду 
```sh
make install
```
> Исполняемый файл будет находится в папке:
MACOS: s21_smart_calc/smart_calc.app/Contents/MacOS
LINUX: s21_smart_calc/

## Запуск

- Из папки SRC выполните следующую команду 
```sh
make launch
```

## Архивация

- Из папки SRC выполните следующую команду
```sh
make dist
```

## Удаление

- Из папки SRC выполните следующую команду
```sh
make uninstall
```
